Bio

Learn more about me

I'm a front-end developer that hand-crafts usable, elegant and effective web sites. I'm currently working for tripsta / airtickets, an Online Travel Agency (OTA). I have over 9 years of experience in front-end technologies and love working with Javascript (ES5 and ES6 / ES2015).

For the last couple of years, I've been enjoying building components using React with various state management tools - redux and MobX state tree - as well as react's extensive ecosystem ( react-router, redux-thunk, redux-form, immutable.js, etc.). I'm also improving my knowledge in functional and reactive programming using libraries such as ramda.js, RxJS, and Bacon.js. I also have hands-on experience with many other Javascript frameworks and libraries such as backbone.js, underscore and lodash, jQuery, axios, moment.js and many more.

On the backend stack, I like working with node.js and express framework, but also have decent knowledge in Ruby and Ruby on Rails. I'm a huge fan of Linux OS - currently using Ubuntu - and git, agile and trying to do as much of TDD (Test-Driven Development) as possible.
